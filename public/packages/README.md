# Package photos — swap in your own

Each package shows the image at `public/packages/<slug>.jpg`. **To use your own photo,
just replace that file** (keep the same name; landscape JPG, ideally ≥1200px wide) and
redeploy — the card, the detail page and the social/WhatsApp preview all pick it up
automatically. No code change needed.
